# Enterprise Plugin

企业级插件，提供代码审查、安全扫描、性能测试等功能。

## 目录结构

```
.claude-plugin/     # 元数据目录
commands/           # 简单 skills（单个 .md 文件）
skills/             # Agent Skills（推荐，目录+SKILL.md）
agents/             # 子智能体
hooks/              # 事件钩子
scripts/            # 辅助脚本
```

## 使用方法

1. 安装插件
2. 配置相关设置
3. 开始使用各项功能

## 功能特性

- 代码审查
- 安全扫描
- 性能测试
- 合规检查
